# Sound Files

This directory contains audio files used in the MUT Hologram Studio application.

## Required Sound Files

### camera-shutter.mp3
A camera shutter/click sound effect played when capturing photos.

**Where to get:**
- Free sound effect websites like [Freesound.org](https://freesound.org) (search for "camera shutter")
- [Zapsplat](https://www.zapsplat.com) (royalty-free sound effects)
- Record your own camera shutter sound

**Requirements:**
- Format: MP3
- Duration: 0.5-1 second
- Volume: Medium (normalized)

**Example searches:**
- "camera shutter"
- "camera click"
- "DSLR shutter"

Once downloaded, place the file here as `camera-shutter.mp3`.
